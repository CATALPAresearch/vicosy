Npm Pakete für client und server installieren:

cd client
npm install
cd ..
npm install

oder

npm run client-install
npm install

Entwicklung (startet Backend-Server auf Port 5000 und Development-Server auf Port 3000):
npm run dev



Deployment:
.env bereitstellen:
MONGO_URI="mongodb://fuh:secure7@ds161710.mlab.com:61710/close-up-collab"
SECRET_OR_KEY="secret"
NODE_ENV="production"
SSL_CERT="path/to/certificate"
PORT=5000

Client production build:
cd client
npm run-script build

Backend starten (liefert im "production" Modus auch die statischen Assets aus):
npm start
